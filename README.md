# Horizon Parser

A structured document language parser built with Bison and Flex.

## Directory Structure

```
horizon/
├── src/           # Source files
│   ├── ast.cpp    # AST implementation
│   ├── ast.h      # AST definitions
│   ├── main.cpp   # Main program
│   ├── parser.y   # Parser grammar (Bison)
│   ├── scanner.l  # Lexer/scanner (Flex)
│   └── token.h    # Token definitions
├── tests/         # Test files
│   └── test.txt   # Test document
└── Makefile       # Build configuration
```

## Building

```bash
make
```

## Running

```bash
make test
# or
./parser
```

## Cleaning

```bash
make clean
```

## Generated Files

The following files are generated by Bison/Flex and should not be edited manually:
- `src/parser.tab.cc`, `src/parser.tab.hh` (from parser.y)
- `src/lex.yy.cc` (from scanner.l)
- `src/location.hh`, `src/position.hh`, `src/stack.hh` (from bison)
